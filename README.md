# Relating-vocalized-mimed-imagined-speech-sEEG
Relating neural representations of vocalized, mimed, and imagined speech using an sEEG dataset of Mandarin Chinese

## Overview

This folder trains and evaluates linear (Ridge) decoders that reconstruct audio
spectrograms from sEEG, and tests how well a decoder trained on one speech
modality (Vocalized, Mimed, or Imagined) generalizes to the other two.

Pipeline: audio -> cortical-model spectrogram, sEEG -> aligned sliding windows,
Ridge regression windows -> spectrogram frames, then comparison of reconstructions
across modalities (including a shuffled-target null model as a control).

## Dependencies

Relies on two sibling packages on `sys.path`:
- `spect_tools` (`SPECT_TOOLS_PATH`) - cortical model spectrogram computation
- `stimReconstruction/gv_per_sentence.py` - audio/sEEG loading, path resolution,
  time alignment helpers

Paths to the audio/sEEG data directories and the cochlear filterbank
(`cochba_v7.mat`) are hardcoded as constants in `stimrcnt_per_sentence.py`.

## Core library

- `stimrcnt_per_sentence.py` - shared utilities used by every script below:
  sliding-window construction, `train_linear_decoder` (per-frequency Ridge),
  `reconstruct_spectrogram`, and the path/parameter constants (window size,
  hop size, channels/frequencies kept, etc.). Its `main()` is a quick sanity
  check on a handful of sentences.

## Generalization experiments

- `stimrcnt_gen.py` / `stimrcnt_gen_envelope.py` - within/across-sentence
  generalization study on full spectrograms vs. their envelopes.
- `lin_ridge_across_modes/linear_ridge_modes.py` - **main experiment.** Trains
  one Ridge decoder per modality (causal windows: past activity predicts the
  current frame) on a sentence-level train split, then evaluates it on its own
  held-out sentences plus the *other two* modalities in full. Supports a
  `--null` flag for shuffled-target control models. Run as:
  `python linear_ridge_modes.py --train_mode {Vocalized,Mimed,Imagined,ALL} [--null 1]`
- `lin_ridge_across_modes/linear_ridge_modes_noncausal.py` - same experiment
  with centered (non-causal) windows, i.e. past + future neural activity
  predicts the current frame.

## Analysis / plotting (consume the saved `.npz` results)

- `analyze_gen.py` - per-sentence correlation, bootstrap CIs, channel
  importance, optional channel-ablation.
- `rank_analysis.py` / `rank_analysis_envelope.py` - sentence/rep
  identification ("rank") matrices from predicted vs. true reconstructions.
- `plot_sentence_rcnt.py` / `plot_envelope_rcnst.py` / `plot_env_corr_dist.py` -
  spectrogram, envelope, and correlation-distribution figures.
- `lin_ridge_across_modes/g_filters.py` - inspects the trained decoder
  coefficients themselves (channel/lag/frequency importance, cross-modality
  decoder similarity).
- `lin_ridge_across_modes/hist_plot.py` / `hist_plot_with_null.py` -
  per-frequency / per-sentence correlation histograms, real vs. null.
- `lin_ridge_across_modes/save_to_mat.py` - exports real-vs-null sentence-level
  envelope correlations to `.mat` for downstream MATLAB use.

## Outputs (not tracked in git, see `.gitignore`)

Each experiment script writes large `.npz` result files plus `.png`/`.pdf`
figures into its own output directory (`generalization_results/`,
`generalization_results_envelope/`, `outputs*/`, `per_sentence_results/`,
`rank_outputs/`). These are regenerated by re-running the corresponding
script and are excluded from version control due to size.
